Le programme suivant a été réalisé dans un cadre pédagogique, il représente le premier travail pratique du 2e semestre en Algorithmique et Structure des Données Dynamiques réalisé par Aouaouche 
Louiza et Mechouek Lounes Section A, Groupe 2 - 2017/2018 Ecole Nationale Supérieure d'Informatique.

Il vise à organiser les inscriptions des athlètes pour les Jeux Olympiques.

	Consignes d'usage :

Veuillez s'il vous plaît, pour tout usage de compilation ou de tests effectuer ces derniers à partir du projet "TP_Final".
Pour l'exécution immediate, un fichier .exe opérationnel est mis à disposition dans .\bin\debug.
Toute modification ou amélioration est permise, il n'y a aucune restriction de droits d'auteurs si ce n'est la conservation des noms des réalisateurs originels ainsi que ce fichier texte.

	Bibliothèques et fichiers sources :

Ce programme utilise 3 bibliothèques :
	*Bibliothèque "logicielle": Contient toutes les procédures précisées dans l'énoncé du TP (A retrouver sur tresoresi.dz). Son code source "TP_Biblio.c" & "TP_Biblio.h".
	*Bibliothèques "graphiques" : Nous nous sommes appuyés sur deux bibliothèques (open-source) de ce type pour réaliser nos graphismes :
					*Bibliothèque "conio": A servi a la réalisation du menu ainsi que la fludification des fonctions logicielles en y ajoutant une interface graphique. (Graphics.c)
					*Bibliothèque "SDL"  : A servi a la réalisation de l'animation de début. Vous trouverez le code source utilisé directement dans le fichier "Aouaouche_Mechouek_TP1_G2.c"
